# Priyanshu AI Chat Frontend

A modern React chat interface powered by Vite, connected to the Priyanshu AI FastAPI backend.

## Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend will run on **http://localhost:5173** and connect to the FastAPI backend on **http://localhost:8000**.
